# ⚙️ E-Commerce Backend (Node.js + Express)

This is the backend API for the e-commerce application.

## 🚀 Features
- REST API for products
- Add product
- Get all products
- Get single product
- Update product
- Delete product

## 🛠️ Tech Stack
- Node.js
- Express.js
- MongoDB (Mongoose)

## 📡 API Endpoints

### Get all products
GET /products

### Get single product
GET /products/:id

### Add product
POST /products

### Update product
PUT /products/:id

### Delete product
DELETE /products/:id

## ▶️ Run Server
```bash
npm install
node server.js
```
